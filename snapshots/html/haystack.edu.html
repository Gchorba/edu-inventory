The filtering software we are using is called DSPAM and is capable of dynamically learning your specific email behavior.

You will need to establish a login user and password in order to check your quarantine box, please go to the following page to create it (you only do this once):
            http://haystack.mit.edu/dspampass.html
use the same login as you login to your system, but choose a different secure password.

To access your quarantine box, you will go to:
            http://haystack.mit.edu/cgi-bin/dspam/dspam.cgi
This will popup a dspam login box for you to enter your user and password you generated
